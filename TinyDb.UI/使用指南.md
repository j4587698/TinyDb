# TinyDb 管理器使用指南

## 🚀 快速开始

### 1. 启动应用程序
应用程序已在后台运行，您应该能看到一个现代化的数据库管理界面。

### 2. 连接数据库

#### 方法一：使用浏览按钮
1. 点击"浏览..."按钮 ✅ **现在应该可以点击了**
2. 应用会自动设置默认路径：`tinydb_manager.db`
3. 点击"连接"按钮

#### 方法二：手动输入
1. 在数据库文件输入框中直接输入文件名（如：`mydatabase.db`）
2. 点击"连接"按钮

### 3. 开始使用
连接成功后，您可以：
- 📊 查看数据库统计信息
- 📁 管理集合（新建、删除）
- 📄 管理文档（增删改查）
- ✏️ 编辑JSON格式的文档内容

## 🔧 界面说明

### 顶部工具栏
- **数据库文件输入框**：输入数据库文件路径
- **浏览按钮**：设置默认数据库路径
- **连接按钮**：连接到数据库（需要输入路径）
- **断开按钮**：断开当前连接
- **数据库信息**：显示连接状态和统计信息

### 主界面三栏布局
1. **左侧 - 集合列表**
   - 显示所有数据库集合
   - 刷新、新建、删除集合

2. **中间 - 文档列表**
   - 显示选中集合的文档
   - 新建、删除文档
   - 表格形式展示文档信息

3. **右侧 - 文档编辑器**
   - JSON格式编辑文档内容
   - 保存修改
   - 显示文档详细信息

### 底部状态栏
- 显示当前操作状态
- 加载进度指示器

## ✨ 功能演示

### 创建测试数据
1. 连接数据库后，点击左侧"新建"创建集合
2. 选择新集合，点击中间"新建文档"
3. 在右侧编辑器中修改JSON内容
4. 点击"保存"按钮

### 查看数据库信息
连接成功后，顶部会显示：
- 数据库名称和文件大小
- 页面使用情况
- 集合数量
- 缓存统计信息

## 🎯 按钮状态说明

### 初始状态（未连接）
- ✅ **浏览按钮**：始终可用
- ⭕ **连接按钮**：需要输入数据库路径后才可用
- ⭕ **断开按钮**：未连接时不可用
- ⭕ **集合管理按钮**：未连接时不可用

### 连接后状态
- ✅ **断开按钮**：可用
- ✅ **集合管理按钮**：可用
- ✅ **文档操作按钮**：选择集合/文档后可用

## 🔍 故障排除

### 如果按钮仍然是灰色的
1. 确保在数据库文件输入框中输入了文件名
2. 检查底部状态栏是否有错误信息
3. 尝试重启应用程序

### 如果连接失败
1. 检查数据库文件路径是否正确
2. 确保有写入权限
3. 查看底部状态栏的错误提示

## 📝 示例用法

1. **创建新数据库**：
   - 输入 `test.db` → 点击"连接" → 点击"新建集合"

2. **添加文档**：
   - 选择集合 → 点击"新建文档" → 编辑JSON → 点击"保存"

3. **查看数据**：
   - 选择集合 → 查看文档列表 → 选择文档查看内容

现在您应该可以正常使用所有功能了！🎉
